SuperEnglishCol Quiz

Este es un proyecto del SENA dedicado a enseñar inglés de manera interactiva. SuperEnglishCol es una iniciativa educativa diseñada para mejorar el aprendizaje de lenguas extranjeras entre los estudiantes de primaria. Este proyecto ha sido desarrollado por Dana Sofia y Gabriel Avila, quienes están comprometidos con brindar herramientas interactivas y efectivas para la educación.

Nuestro objetivo es preparar a los estudiantes de una manera divertida y educativa para la prueba ICFES, con preguntas adaptadas específicamente para niños de primaria. ¡Esperamos que disfrutes aprendiendo con nosotros!
